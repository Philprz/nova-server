Approche technique – Pilote
1) Connecteurs
   ◦ SAP Business One (requêtes API ou export CSV)
   ◦ Lecture Excel/PDF (parsing automatisé)
   ◦ Connexion Office365 (API Graph, sans SSO pour le pilote)
   ◦ Intégration future : API DHL / UPS
2) Moteur de consolidation
   ◦ Développé en Python / FastAPI (module NOVA ou microservice dédié)
   ◦ Calcul automatique : coût fournisseur + transport + marge
   ◦ Stockage temporaire en base ou cache (PostgreSQL / Redis)
3) Interface utilisateur
   ◦ Interface NOVA ou mini-dashboard pour consulter et valider les devis
   ◦ Historisation automatique des devis générés
4) Chiffrage du pilote
   ◦ Objectif : POC fonctionnel avec 3 sources (SAP, Excel/PDF, Office365)
